# PaletoPDExteriorUpgrade
Adds a automatic gate and a new helipad to Paleto Bay Police Department. (YMAP)

HOW TO INSTALL:
1. Install "PaletoPD"
2. Add "PaletoPD" to your resources folder
3. Add "start PaletoPD" to your server.cfg
4. Restart your server
